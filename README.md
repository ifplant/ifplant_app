# `IfPlant` : 당신의 반려식물을 움직여보세요

한번 구매하면 처리하기 어려운 `반려식물`을 가상으로 `원하는 장소`에 미리 배치해보면서 어울리는지 판단해보고, 합리적으로 구매할 수 있게 도와줄게요. <br/>
